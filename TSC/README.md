# tcards-site — СЫТНАЯ ДОСТАВКА / Технологические карты

## Что это
Статический сайт-каталог технологических карт (GitHub Pages).
Есть:
- Фильтры: Доставка / Категория
- Поиск по названию
- Сортировка по названию (A→Z / Z→A)
- Избранное (localStorage)
- Страница блюда + кнопки: Копировать / Печать
- Админка: добавить/редактировать/удалить (фото всегда заглушка)
- Кнопка «Сохранить в GitHub» — коммитит data/dishes.json через GitHub API (токен в коде)

## Где править токен и репозиторий
Файл: `js/common.js`

Заменить:
- `GITHUB_TOKEN` — ваш токен
- `GITHUB_OWNER` — ваш GitHub username
- `GITHUB_REPO` — имя репозитория

## Про фото
По умолчанию используется `assets/photos/placeholder.jpg`.
Чтобы поставить реальное фото: загрузите файл в `assets/photos/` и поменяйте поле `photo` у блюда в `data/dishes.json` (или экспортируйте/отредактируйте).

## Локальный запуск
Нужен локальный сервер (иначе fetch может не работать при file://):
```bash
cd tcards-site
python -m http.server 8000
```
Открыть: http://localhost:8000
