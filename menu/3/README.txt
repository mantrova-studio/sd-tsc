Папка 3 — шаблон печатного меню Kraken.

Как поставить в архив/репозиторий:
1. Скопируйте папку menu/3 в корень сайта рядом с menu/1 и menu/2.
2. Откройте /menu/3/index.html.
3. Пароль редактора: 5555.
4. Кнопка «Печать / Save as PDF» или «Скачать PDF» открывает печать браузера, где можно сохранить PDF.
5. Шрифты не дублируются в папке 3: index.html берет их из ../2/fonts. Поэтому не удаляйте menu/2/fonts.
6. Правки меню сохраняются в браузере через localStorage. Для переноса на другой компьютер используйте блок «Резервная копия / JSON».
